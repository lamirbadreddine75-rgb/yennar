
# تحويل تطبيق بايثون إلى تطبيق أندرويد

هذا المشروع يوضح كيفية تحويل تطبيق بايثون يعمل على الحاسوب إلى تطبيق أندرويد باستخدام Kivy وKivyMD.

## الملفات في المشروع

- `gui_modern.py`: التطبيق الأصلي الذي يعمل على الحاسوب باستخدام Tkinter.
- `android_app.py`: نسخة التطبيق المحولة ليعمل على أندرويد باستخدام Kivy وKivyMD.
- `buildozer.spec`: ملف إعدادات Buildozer لتجميع التطبيق إلى ملف APK.
- `requirements.txt`: المكتبات المطلوبة لتشغيل التطبيق.

## خطوات تحويل التطبيق إلى تطبيق أندرويد

### 1. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 2. تشغيل التطبيق على الحاسوب (للاختبار)

```bash
python android_app.py
```

### 3. تجميع التطبيق إلى ملف APK

```bash
buildozer android debug
```

### 4. تثبيت التطبيق على جهاز أندرويد

```bash
adb install bin/myapp-0.1-armeabi-v7a-debug.apk
```

## متطلبات إضافية

لتجميع التطبيق إلى ملف APK، ستحتاج إلى تثبيت:

1. **Android SDK**: يمكنك تثبيته عبر Android Studio.
2. **Android NDK**: مطلوب لتجميع الكود الأصلي.
3. **Java Development Kit (JDK)**: مطلوب لعملية التجميع.

## ملاحظات هامة

- تأكد من أن متغيرات البيئة (JAVA_HOME, ANDROID_HOME, ANDROID_NDK_HOME) معدة بشكل صحيح.
- عملية التجميع الأولى قد تستغرق وقتًا طويلاً لأنها تقوم بتنزيل جميع الاعتماديات.
- يمكنك تعديل ملف buildozer.spec لتخصيص إعدادات التطبيق مثل الأيقونة، شاشة البداية، الصلاحيات، إلخ.

## استكشاف الأخطاء وإصلاحها

إذا واجهت مشاكل أثناء عملية التجميع، تحقق من:

1. إصدارات Android SDK و NDK المتوافقة.
2. مسارات المتغيرات البيئية.
3. صلاحيات الوصول إلى المجلدات.
4. توافق إصدارات المكتبات مع بعضها البعض.

## بدائل أخرى

باستخدام Kivy وKivyMD، يمكنك أيضًا:

1. **BeeWare**: مجموعة أدوات أخرى لتحويل تطبيقات بايثون إلى تطبيقات أصلية.
2. **Chaquopy**: مكون إضافي لـ Android Studio يتيح استدعاء كود بايثون من تطبيقات Java/Kotlin.
3. **PySide/PyQt for Android**: لتحويل تطبيقات PyQt إلى تطبيقات أندرويد.
